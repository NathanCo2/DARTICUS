Detailed Description

This file is the main code used to run the thermal heating seeking nerf gun turrent, D.A.R.T.I.C.U.S.

D.A.R.T.I.C.U.S stands for Dueling Autonomous infraRed Tracking IronClad Unrivaled Sentry. The following program contains an inter-task shared variable using four shares. This code sets up a STM32L476 intializing two DC motors using PID control with encoder readers and a mlx90640 camera.
